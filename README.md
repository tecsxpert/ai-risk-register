# AI Risk Register

This project is a multi-service application consisting of a Spring Boot backend, a Flask AI service, and a React frontend.

## Project Structure
- `backend/`: Spring Boot REST API
- `ai-service/`: Flask Python microservice for AI integration
- `frontend/`: React + Vite frontend

## Getting Started
1. Configure `.env` based on `.env.example`.
2. Run with Docker: `docker-compose up --build`
